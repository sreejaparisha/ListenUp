💡 CUSTOMIZATION QUICK TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Change Accent Color:
  Edit: assets/css/variables.css
  Variables: --accent-primary, --accent-secondary, --accent-hover

Add New Component:
  1. Create styles in appropriate CSS file
  2. Use CSS variables for colors/spacing
  3. Use BEM naming convention
  4. Add theme support

Modify Typography:
  Edit: assets/css/variables.css
  Scale: 12px to 48px with 9 size levels
